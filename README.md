# Expense Tracker (CLI)

A simple command-line expense tracker built using Python.

## Features
- Add daily expenses
- Store data in CSV file
- View monthly category-wise report
- Generate pie chart and bar chart using matplotlib

## Requirements
- Python 3
- matplotlib

## How to Run

pip install matplotlib
python main.py


## Files

main.py – CLI menu

tracker.py – Expense logic

expenses.csv – Auto-generated data file
